# .prompt.md — Prompt templates for common tasks

Use these short templates when asking an AI agent to act on the codebase. Replace placeholders in `{{double_braces}}`.

1) Fix failing tests

```
Task: Run tests and fix failures
Run: `uv sync && uv run pytest -q`
Output: Show failing tests and stack traces
Goal: Implement minimal code or test fixes so all tests pass. Add tests if a bugfix is introduced.
```

2) Add feature

```
Task: Implement feature
Context: Files touched: {{file1.py, file2.py}}
Requirements: {{concise bullet list of behavior}}
Tests: Provide unit tests under `tests/` that validate behavior
```

3) Refactor module

```
Task: Refactor `{{module_path}}` to improve readability
Constraints: Preserve public API and existing tests
Output: Modified source + updated tests and changelog entry if needed
```

4) Documentation update

```
Task: Update docs for `{{feature}}`
Files: Prefer updating `docs/` and `README.md` links, do not duplicate large sections
```
