# .agent.md — Short agent policy for covdrugsim

Purpose: Provide a compact, machine-friendly set of rules for AI agents working on this repository.

- **Minimal edits only:** Make focused changes with tests. Avoid broad refactors unless requested.
- **Do not change release/version metadata** (`pyproject.toml:project.version`) without explicit instruction from maintainers.
- **Run tests before proposing merges:** Use `uv sync` then `uv run pytest -q` (or `python -m pytest tests/`). Fix or add tests for code changes.
- **Respect documentation:** Link to existing docs in `docs/`, `README.md`, and `CONTRIBUTING.md` rather than copying their contents.
- **CI awareness:** If suggesting CI changes, inspect `.github/workflows/` and update workflows only after discussing with maintainers.
- **Environment notes:** Project targets Python >=3.10 (see `pyproject.toml`). Use virtual environments or `uv`.

Files to check first: `pyproject.toml`, `README.md`, `CONTRIBUTING.md`, `docs/`, `tests/`, `src/covdrugsim`.
